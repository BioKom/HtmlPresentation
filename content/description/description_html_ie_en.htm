This is the HTML version of the presentation.
Html version control (the currently displayed):
  • Click the mouse on the page number at a single slide to show the slide.
  • The actual slide can be accessed using the space bar, enter key or arrow to the right.
  • You can use the number keys on the first and key letter on the second keyboard line to get to the first 21 slides (by numbers) directly.
Control of the presentation mode (display of a single slide):
  • With the '<' button you return to this (HTML) view.
  • The next slide can be accessed using the space bar, enter key or arrow to the right.
  • With the "F11" key you can switch to full screen mood and back to normal screen.
  • You can adapt the slide to the screen size with the key combination [Strg] [+] and [Strg] [-] .
  • Left arrow shows the previous slide.
  • Up arrow shows the first slide.
  • Down arrow shows the last slide.
  • You can use the number keys on the first and key letter on the second keyboard line to get to the first 21 slides (by numbers) directly.
